Prerequisitos

¿Qué necesitás?

Para Claude Code solo necesitás Node.js. Para subir a GitHub, además necesitás Git.

🟢

Node.js + npm

El motor de JavaScript. Lo instalamos via nvm para evitar todos los errores de permisos.

🔑

Cuenta Anthropic

Para el login de Claude Code. Sirve cuenta de Claude Pro o una API Key gratis.

🐙

Git (para GitHub)

Solo si querés subir código. Mac lo trae preinstalado. Windows: descargar de git-scm.com

⚠️ Por qué NO usar nodejs.org directo en Mac

El instalador de nodejs.org instala Node en /usr/local (requiere permisos de root) → npm install -g falla. Con nvm Node se instala en tu carpeta de usuario → cero permisos, cero errores.

macOS — método correcto

Instalación en Mac con nvm

  1. Abrí la Terminal
    Spotlight → Cmd + Espacio → escribí "Terminal"
  2. Instalá nvm
    Pegá el comando curl de la derecha y ejecutalo. Reiniciá la terminal después.
  3. Instalá Node con nvm
    nvm install --lts — instala la última versión estable.
  4. Instalá Claude Code
    npm install -g @anthropic-ai/claude-code — sin sudo, sin errores.
  5. Iniciá sesión
    claude — login con tu cuenta Anthropic.
Terminal — macOS

Windows

Instalación en Windows

  1. Descargá Node.js
    Entrá a nodejs.org → descargá el instalador LTS (.msi)
  2. Instalá Node
    Ejecutá el .msi y seguí los pasos. Marcá "Add to PATH".
  3. Abrí PowerShell como Administrador
    Clic derecho en el menú inicio → "Windows PowerShell (Admin)"
  4. Instalá Claude Code
    npm install -g @anthropic-ai/claude-code
  5. Iniciá sesión
    claude — hacé login con tu cuenta Anthropic.
PowerShell — Windows (Admin)
# 1. Verificar Node node -v # → v22.x.x ✓ OK # 2. Instalar Claude Code npm install -g @anthropic-ai/claude-code # Si da error de ejecución de scripts: Set-ExecutionPolicy RemoteSigned -Scope CurrentUser # 3. Verificar y abrir claude --version claude
Errores frecuentes

Errores comunes y cómo solucionarlos

Error 1 — Mac
EACCES: permission denied, mkdir '/usr/local/lib/node_modules'
Fix Tenés Node instalado desde nodejs.org (en /usr/local). Desinstalalo y usá nvm en su lugar: curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.40.3/install.sh | bash → reiniciá la terminal → nvm install --lts
Error 2 — Mac / Linux
nvm: command not found (después de instalar nvm)
Fix Cerrá la terminal completamente y volvé a abrirla. nvm modifica el archivo ~/.zshrc y eso solo toma efecto en una terminal nueva.
Error 3 — Windows
running scripts is disabled on this system
Fix Abrí PowerShell como Administrador y corré: Set-ExecutionPolicy RemoteSigned -Scope CurrentUser — luego volvé a intentar.
Error 4 — Mac / Windows
brew: command not found / gh: command not found
Fix Homebrew y gh CLI no son necesarios para instalar Claude Code. Son herramientas opcionales. Podés ignorar este error.
Fix permisos — Mac

Si npm install -g da error de permisos

Alternativa al nvm: redirigir la carpeta global de npm a tu usuario. Cero sudo, cero root.

❌

Nunca hagas esto

sudo npm install -g @anthropic-ai/claude-code

Instala con permisos de root → futuros updates fallan → más errores en cadena.

✅

Hacé esto en su lugar

# 1. Crear carpeta global en tu usuario mkdir ~/.npm-global npm config set prefix '~/.npm-global' # 2. Agregar al PATH — pegá esto en ~/.zshrc export PATH=~/.npm-global/bin:$PATH # 3. Recargar la terminal source ~/.zshrc # 4. Instalar sin sudo npm install -g @anthropic-ai/claude-code added 1 package ✓
GitHub

Para subir código a GitHub

Claude Code no necesita Git. Pero si querés subir tu proyecto a GitHub, sí vas a necesitarlo.

🍎 macOS

Git ya viene incluido

Mac tiene Git preinstalado. Verificá con git --version. Si pide instalar Xcode Command Line Tools, aceptá.

git --version git version 2.x.x ✓ # Configurar tu identidad git config --global user.name "Tu Nombre" git config --global user.email "tu@email.com"
🪟 Windows

Descargar Git for Windows

Entrá a git-scm.com y descargá el instalador. Durante la instalación dejá todas las opciones por defecto.

# Después de instalar, en PowerShell: git --version git version 2.x.x ✓ # Configurar tu identidad git config --global user.name "Tu Nombre" git config --global user.email "tu@email.com"
💡 Para hacer push a GitHub

La primera vez que hagas git push, GitHub va a pedir tu usuario y contraseña. En la contraseña no pongas tu password, sino un Personal Access Token que generás en github.com → Settings → Developer Settings → Tokens.

Verificación

Verificá que todo funciona

  • Node instalado: node -v muestra versión
  • npm instalado: npm -v muestra versión
  • Claude instalado: claude --version muestra versión
  • Claude abre correctamente: claude sin errores
  • Login completado con tu cuenta Anthropic
💡 Tip

Siempre abrí Claude desde la carpeta de tu proyecto:
cd mi-proyecto && claude
Claude Code va a leer el contexto de tu código automáticamente.
