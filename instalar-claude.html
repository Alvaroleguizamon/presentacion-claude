Prerequisitos

¿Qué necesitás antes?

Solo una cosa es obligatoria: Node.js. Todo lo demás se instala como consecuencia.

🟢

Node.js

El motor que corre JavaScript. Claude Code lo necesita. Obligatorio.

📦

npm

El instalador de paquetes. Viene incluido con Node.js automáticamente.

🔑

Cuenta Anthropic

Para el login. Podés usar cuenta de Claude Pro o una API Key de console.anthropic.com

💡 Aclaración importante

No necesitás Homebrew, ni gh CLI, ni git para instalar Claude Code. Solo Node.js.

macOS

Instalación en Mac

  1. Abrí la Terminal
    Buscá "Terminal" en Spotlight (Cmd + Espacio)
  2. Verificá si ya tenés Node
    Escribí: node -v — si aparece un número, listo.
  3. Si no tenés Node, instalalo
    Entrá a nodejs.org y bajá el instalador LTS para Mac.
  4. Instalá Claude Code
    npm install -g @anthropic-ai/claude-code
  5. Iniciá sesión
    claude — te va a pedir hacer login con tu cuenta.

Windows

Instalación en Windows

  1. Descargá Node.js
    Entrá a nodejs.org → descargá el instalador LTS (.msi)
  2. Instalá Node
    Ejecutá el .msi y seguí los pasos. Marcá "Add to PATH".
  3. Abrí PowerShell como Administrador
    Clic derecho en el menú inicio → "Windows PowerShell (Admin)"
  4. Instalá Claude Code
    npm install -g @anthropic-ai/claude-code
  5. Iniciá sesión
    claude — hacé login con tu cuenta Anthropic.

Errores frecuentes

Errores comunes y cómo solucionarlos

Error 1 — Mac / Linux
npm: command not found
Fix No tenés Node instalado. Entrá a nodejs.org y descargá el instalador LTS.
Error 2 — Mac / Linux
EACCES: permission denied, mkdir '/usr/local/lib/node_modules'
Fix No uses sudo. En cambio, corré: mkdir ~/.npm-global && npm config set prefix '~/.npm-global' y agregá export PATH=~/.npm-global/bin:$PATH a tu ~/.zshrc o ~/.bashrc. Luego reiniciá la terminal.
Error 3 — Windows
running scripts is disabled on this system
Fix Abrí PowerShell como Admin y corré: Set-ExecutionPolicy RemoteSigned -Scope CurrentUser
Error 4 — Mac / Linux
brew: command not found
Fix Homebrew no es necesario para instalar Claude Code. Podés ignorar este error o instalarlo desde brew.sh si querés tenerlo disponible para otras cosas.
Permisos

¿Por qué no usar sudo?

Muchos guías te dicen sudo npm install -g ... pero esto crea problemas de permisos a largo plazo.

❌

No hacer esto

sudo npm install -g @anthropic-ai/claude-code

Instala con permisos de root → futuros updates fallan → más errores.

✅

Hacer esto en su lugar

# Crear carpeta global de npm mkdir ~/.npm-global npm config set prefix '~/.npm-global' # Agregar al PATH (en ~/.zshrc o ~/.bashrc) export PATH=~/.npm-global/bin:$PATH # Recargar la terminal source ~/.zshrc # Ahora instalar sin sudo npm install -g @anthropic-ai/claude-code
Verificación

Verificá que todo funciona

  • Node instalado: node -v muestra versión
  • npm instalado: npm -v muestra versión
  • Claude instalado: claude --version muestra versión
  • Claude abre correctamente: claude sin errores
  • Login completado con tu cuenta Anthropic
